# 自己教師あり学習

ラベルなしデータから、口実タスク（pretext task）を通じて転移可能な視覚表現を学ぶ枠組み。大規模な視覚事前学習は大きく2系統に分かれる: **(1) Masked Image Modeling（MIM）** と **(2) Contrastive Learning（CL）**。[[entities/Sapiens2]] はこの両者を**1つの統一目的**に結合した点が中核の貢献である。

## Masked Image Modeling（MIM）

入力パッチの一部をマスクし、それを復元させることで表現を学ぶ。NLP のマスク言語モデリングに倣うが、言語のトークンが自己完結的に意味を持つのに対し、**画像パッチは文脈依存で意味が曖昧**な点が異なる。MIM 目的はしばしば入力トークンの**圧縮**と解釈される。

- **BEiT**: dVAE トークナイザでパッチを離散化し、マスクパッチのコードブック index を予測。
- **MAE（Masked Autoencoder, He et al. 2022）**: パッチの **75%** をマスクし、欠損画素を直接再構成。エンコーダは可視トークンのみ処理し、軽量デコーダが全パッチを復元。前世代 Sapiens はこの MAE 中心だった。
- 特性: 信号・空間詳細・**色などの測光手掛かりを保つ**（画素にアンカーされる）が、意味の表現には中〜高程度の教師を要する。

## Contrastive Learning（CL）

同一画像の異なる拡張ビューを近づけ（positive）、異なる画像を遠ざける（negative）ことで、**インスタンス不変な意味**を注入する。BYOL / SimCLRv2 / MoCov3 / DINO など。

- 特性: 強い意味理解を獲得するが、**グローバル不変性目的**は密予測（細かい空間詳細・測光忠実度が重要）で劣りやすい。
- **表現ドリフト**のリスク: 強い拡張（特に外観 augmentation）が teacher/student を真の観測から乖離させ、色など人物密タスクに不可欠な手掛かりを損なう。

## ハイブリッド（MIM + CL）

両者は相補的なので結合が自然。先行研究:
- **iBOT**: MIM と CL 風自己蒸留を結合。画素やコードワードを再構成せず、マスク目的で student/teacher 特徴を整合（JEPA / v-JEPA2 と同系）。
- **DINOv2**: iBOT 目的を主戦略として採用。**DINOv3** はレシピ改良でさらにスケール。
- ただし**潜在空間目的**は抽象ドリフトのリスク: 表現が観測（画像/文）にアンカーされず、色など密予測に不可欠な手掛かりを捨てる lossy 圧縮になりうる。
- **CMAE** は MAE + CL を探索したが、主に分類で評価。

### Sapiens2 の統一目的

[[entities/Sapiens2]] は **画像アンカーの MAE 目的 + 意味的な CL 目的**を billion-scale・複数の人物タスクで結合する:

```
L = L_MAE + λ · L_CL
```

- **L_MAE**: 各ビューでマスク（マスク率 r）。エンコーダ Φ_enc は可視トークンのみ処理、デコーダ Φ_dec が全パッチ再構成。正規化ターゲットに対しマスクトークン上で MSE を平均。
- **L_CL**: DINOv3 ベースの **student–teacher**。teacher は student と同一アーキで非学習、パラメータは student の **EMA**。各ビューの [CLS] 埋め込みを Φ_cls で K logits に射影し softmax。**global↔global / global↔local** の positive ペア集合 S 上で teacher→student クロスエントロピー H(q,p) を平均（同一ビュー同士・local↔local は除外）。
- **マルチビュー**: global 2 + local 4 crop。**global ビューには color augmentation を使わない**（MAE 再構成のターゲットが汚れるため）。これが CL 系の表現ドリフトを抑える鍵。
- 損失重み（1B）: MAE 1.0 / CLS 0.4 / **KoLeo 0.04**。teacher EMA 0.992、student 温度 0.1、teacher 温度 0.065→0.07。

役割分担は明快で、**L_MAE が低レベル詳細（テクスチャ・色）を、L_CL が人物意味**を担う。これにより、ゼロショット/few-shot/全教師の各レジームと広範な密予測に転移する表現が得られる（dense probing で DINOv3 等を上回る）。

### マスキング戦略
高解像度向けに **blockwise / patchwise 混合**マスク（blockwise 確率 0.4）、**75% マスク率**、patch 16。1024×768（64×48=3072 patch）で約 2304 patch をマスクし、MAE を正則化しつつ CL に十分な文脈を残す。

## 4K 向け階層型アテンション

4K 入力では、先頭 K 層で **windowed self-attention**（局所テクスチャ・境界）→ [CLS]-guided pooling で空間ストライド √ω だけダウンサンプル → 後続 L 層で **global attention**（大域文脈）。**局所段の後にトークンをマスク**できるため MAE スタイルのスパース事前学習と両立し、畳み込みバックボーンが必要とする masked convolution の漏洩問題を回避できる。機構の基礎は [[concepts/Self-Attention]]。
