# GradCAM

**Gradient-weighted Class Activation Mapping**。CNNの予測に対し、入力画像のどの領域が判断に寄与したかを粗いヒートマップで可視化する**事後的（post-hoc）な解釈性手法**（Selvaraju et al., 2017）。

## 仕組み

対象クラスのスコアを、最終畳み込み層の特徴マップに対して逆伝播し、その勾配を空間方向に大域平均プーリングして各特徴マップの重みを得る。重み付き和をReLUに通して、入力画像にオーバーレイ可能な粗い局在化マップ（重要領域ほど高温）を生成する。

特徴:
- **アーキテクチャ非依存**: 構造を変更せず、学習済みの任意のCNNに後付けで適用できる
- **クラス弁別的**: 特定の予測クラスに対する根拠領域を示す
- **粗い解像度**: 最終畳み込み層の空間解像度に依存し、ピクセル精度ではない

## 用途 — 説明可能AIとしての検証

予測の根拠を可視化することで、モデルが**意味のある特徴**に注目しているか（=ショートカット学習やデータリークではないか）を検査できる。医療画像では、臨床的に妥当な領域を見ているかの確認に使われ、臨床現場への統合に向けた信頼性・説明可能性の裏付けとなる。

## 縫合品質分類での適用

[[sources/縫合品質分類における転移学習の有効性評価]]では、ImageNet事前学習CNNによる縫合品質2値分類のモデルにGradCAMを適用。ヒートマップは**ステッチ・結び目・組織縁**に注目していることを示し、外科的品質評価の基準（ステッチの均一性・組織の接合）と整合した。

一方、最難の[[sources/縫合品質分類における転移学習の有効性評価|COOS（連続縫合）]]では縫合線そのものが組織に埋もれて不明瞭なため、GradCAMで縫合線への注目は確認できても識別が困難であり、これがドメインシフトによる性能低下の裏付けとなった（[[concepts/転移学習]]）。

## アテンションベース解釈性との対比

GradCAMは**学習済みCNNへの事後付与**である点で、モデル自身の機構が解釈を生む[[concepts/時間アテンションプーリング]]（[[entities/CNN-LSTMハイブリッドモデル]]のアテンション重み）や[[concepts/Self-Attention]]とは性質が異なる。

| | GradCAM | アテンション重み |
|---|---|---|
| 性質 | 事後的（勾配ベース） | モデル内在的 |
| 対象 | 空間（画像領域） | 時間/系列・空間いずれも可 |
| 適用 | 任意の学習済みCNN | アテンション機構を持つモデル |

[[concepts/外科技術自動評価]]では、こうした解釈性が**形成的フィードバック**（どこ・いつが良い/悪いかを術者に示す）の鍵とされる。
