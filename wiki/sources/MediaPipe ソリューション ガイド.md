## 要約

**MediaPipe Solutions** は、アプリケーションに AI/ML 手法を迅速に組み込むためのライブラリ・ツール群。すぐに組み込め、ニーズに合わせてカスタマイズでき、複数の開発プラットフォーム（Android / Web / Python / iOS）で動作する。MediaPipe オープンソースプロジェクト（github.com/google/mediapipe）の一部であり、ソリューションコードをさらにカスタマイズ可能。

## スイートの構成要素

| 要素 | 役割 |
|---|---|
| **MediaPipe Tasks** | ソリューションをデプロイするためのクロスプラットフォーム API とライブラリ |
| **MediaPipe モデル** | 各ソリューションで使える事前トレーニング済み・即実行可能なモデル |
| **MediaPipe Model Maker** | 自前データでソリューションのモデルをカスタマイズ（転移学習） |
| **MediaPipe Studio** | ブラウザ上でソリューションを可視化・評価・ベンチマーク |

前2つがコア機能（デプロイ）、後2つがカスタマイズ・評価ツール。

## 従来（レガシー）ソリューションの移行

2023-03-01 をもって一部レガシーソリューションのサポートを終了。多くは新 MediaPipe ソリューションへアップグレードされた。

| 旧ソリューション | 移行先 |
|---|---|
| 顔検出 | 顔検出（新） |
| Face Mesh / Iris | 顔のランドマーク検出 |
| 手（Hands） | **手のランドマーク検出** |
| ポーズ（Pose） | **ポーズのランドマーク検出** |
| 総合的（Holistic） | ホリスティックランドマーク検出 |
| セルフィーのセグメンテーション / 髪のセグメンテーション | 画像セグメンテーション |
| オブジェクト検出 | オブジェクト検出（新） |

**サポート終了**（移行先なし）: ボックスの追跡、インスタントモーショントラッキング、Objectron、KNIFT、AutoFlip、MediaSequence、YouTube 8M。レガシーのコードリポジトリとビルド済みバイナリは現状のまま引き続き提供される。

## キーポイント

- MediaPipe Solutions は「事前学習モデル + クロスプラットフォーム API + カスタマイズ/評価ツール」を束ねた**オンデバイス ML のためのスイート**
- ビジョン・テキスト・音声・生成 AI を横断的にカバーし、4 プラットフォームで一貫した API を提供
- 旧来の「手」「ポーズ」ソリューションは、それぞれ[[concepts/手検出・ローカライゼーション|手]]・ポーズの**ランドマーク検出**へ再編された

## 本Wikiでの位置づけ

MediaPipe の **手のランドマーク検出** は、研究領域の[[concepts/3D手姿勢推定]]・[[concepts/手検出・ローカライゼーション]]で「軽量な既存ベースライン」として参照される。[[sources/WiLoR]] の手検出ベンチマーク（Table 1）では MediaPipe は 25Mb / 25 FPS で mAP が低く、in-the-wild のマルチハンド検出では研究 SOTA（WiLoR Proposed-M）に大きく劣ると報告されている。一方で MediaPipe は即デプロイ可能・多プラットフォーム対応という実用面での強みを持つ。
