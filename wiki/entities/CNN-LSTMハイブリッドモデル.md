# CNN-LSTMハイブリッドモデル

[[sources/開放手術スキルの時空間特徴ML評価]]（Alipour et al. 2026）が開放手術スキル分類のために構築した動画分類モデル。**空間特徴抽出（CNN）と時間モデル化（LSTM）を分離**し、両者を直列に接続するハイブリッド構成をとる。3D CNNや動画Transformerが時空間を単一ネットワークで処理するのに対し、本モデルは2D CNNで各フレームを符号化してから時系列を別ネットワークで処理する。

## パイプライン全体

```
入力フレーム (512×512×3)
   │  1 fps サンプリング → 300フレーム/動画
   ▼
fine-tuned ResNet50（分類層除去）          ← 空間特徴抽出
   │  特徴テンソル 314 × 300 × 2048
   ▼
双方向LSTM（BiLSTM）                        ← 時間特徴
   │
単一ヘッド ソフトアテンション               ← 重要フレームの加重和
   │
全結合層 + 正則化
   ▼
softmax → {novice, intermediate, proficient}
```

（[[sources/開放手術スキルの時空間特徴ML評価]] Figure 3 に対応）

## 構成要素

### 空間: fine-tuned ResNet50

[[entities/ResNet50]]を空間特徴抽出器として使用。[[entities/AIxSuture データセット]]のフレームで2フェーズfine-tuning（[[concepts/転移学習]]）した後、分類層を除去して各フレームを2048次元ベクトルに写像する。抽出特徴はキャッシュして冗長計算を回避。

### 時間: 双方向LSTM

[[concepts/LSTM]]のゲート機構（入力/忘却/出力ゲート）で、時系列にわたり重要な変化を選択的に保持・破棄する。**双方向**化により前向き・後ろ向き両方向から系列を処理し、手術タスク全体に及ぶ依存関係を捕捉する。採用理由は ①拡張時間系列の分析が本質的に必要、②長系列の選択的記憶で効率的/非効率的ワークフローを弁別、③双方向で全体文脈を確保。

### アテンション: 時間アテンションプーリング

LSTM出力に**単一ヘッドのソフトアテンション**（[[concepts/時間アテンションプーリング]]）を重畳し、動画中の重要な瞬間に動的に重みを割り当てる:

`e_t = tanh(x_tᵀ W + b_t)` → `α_t = softmax(e_t)` → `output = Σ α_t · x_t`

これにより重要フレームに注目しつつ局所・大域の時間文脈を維持する。アテンション重みはモデルの判断根拠の可視化にも使われ（解釈性）、針把持・初期位置決め・縫合遷移点が高い重みを受けることが示された。

### 分類ヘッド

アテンション出力を全結合層（正則化付き）で段階的に変換し、softmaxで3クラスの確率に写像する。

## 訓練

- **時間サンプリング**: 動画を5分（300秒）に標準化、1 fps（30フレームごと）で300フレーム固定、ウィンドウ重複なし
- **5-fold交差検証**（80%訓練データ上）: 各fold 約64% train / 16% val → 5モデル
- **ハイパラ探索**: グリッドサーチ + ベイズ最適化。学習率 1e-6〜1e-3、dropout 0.1〜0.5、バッチサイズ 8〜32、BiLSTMユニット 64〜512
- 適応学習率、early stopping

## 性能（[[entities/AIxSuture データセット]]、3クラス）

| 指標 | 値 |
|---|---|
| マクロ平均F1 | **0.8204** |
| F1: Novice / Intermediate / Proficient | 0.9126 / 0.6286 / 0.9200 |
| クラス別精度（abstract） | 90.1% / 65.7% / 86.3% |
| 5-fold CV AUC | 0.95 ± 0.009 |
| 推論時間 | 約10.2 ± 0.4 秒/動画 |

[[sources/AIxSuture]]ベンチマークの[[entities/I3D]]（マクロF1 0.69）・[[entities/Video Swin Transformer]]各種（0.63〜0.72）を全スキルレベルで上回る。混同は隣接クラス間に集中し、順序性を保持。

## 長所と短所

- **長所**: 空間と時間を分離するためモジュール性が高い。3D CNN/動画Transformerよりメモリ効率が良い。アテンション重みで解釈可能
- **短所**: ResNet50のフレーム単位処理 + LSTMの逐次処理 + 各timestepのアテンション計算で計算コストが大きく、リアルタイム評価が困難。小規模施設ではGPU基盤が制約に
